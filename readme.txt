== Description ==

With this widget its easy to have relations between posts, depending on their category. 

You can create category-relationships on back-end and have an easy widget on front-end. See or remove existing relations between the current post or add a new.

* All actions are user-role-controlled. It means only users who are allowed to edit the current post, are allowed to edit the relation.
* The relations are stored in the WordPress database
* Easy undone deleting changes


 Example:

 Create a relationship between the categories movies and actors. On front-end add actors playing in a movie.


== Installation ==

1. Upload the plugin to your `/wp-content/plugins/` directory, activate and use as a sidebar-widget.
2. Configure the relations in WordPress backend on -> Settings -> Post Relation Widget
